# BBB-music
## Scripts
The project is containing sripts to compile music sheets into instrument folders.
Moreover it is possible to identify which are missing OR just do not comply with
the underlying naming convention.

## Structure
Assumed structure is a folder containing all music titles with their instrument sheets in sub-folders.
E.g.

- music
   - title 1
   - title 2
      - title2 - instrument1.pdf
      - title2 - instrument2.pdf
      - ...

## Naming convention

*  Die Scans sollten als 300 dpi Schwarz/Weiss-Scans im pdf Format vorliegen

*  Titelname wird in "Camel Case" geschrieben, d.h. jedes Wort im Titel beginnt mit Grossbuchstaben
   Beispiel: "In The Mood"

*  Falls es mehrere Arrangements für ein St¸ck gibt, wird dem Titel in Klammern eine weitere Bezeichnung hinzugefügt 
   Beispiel: "In The Mood (Tyzik)"

*  Die Scans der einzelnen Stimmen werden ebenso benannt mit der Stimmenbezeichnung in Grossbuchstaben als Zusatz, 
   getrennt durch Leerzeichen und Bindestrich " - "
   Beispiel "In The Mood (Tyzik) - TPT1"
